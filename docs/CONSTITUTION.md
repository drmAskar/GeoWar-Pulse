# GeoWar Pulse - Constitution

## Core Principles

1. **First-Release Focus**
   - Focus on a working MVP before adding new features
   - Ship early, iterate often

2. **Data-Driven**
   - All risk scores depend on real data, not guesses
   - Use reliable sources and verify information

3. **Transparency**
   - Display data sources and confidence levels to the user
   - No black boxes - users should understand how scores are calculated

4. **Real-time First**
   - Instant updates are more important than perfect design
   - Prioritize speed and freshness over aesthetics

5. **Security & Privacy**
   - Protect sensitive data
   - Follow best practices for data handling
