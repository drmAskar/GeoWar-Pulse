# Roadmap

## Phase 1 (MVP, 2–3 weeks)
- Source catalog + ingestion contracts
- Initial feature extraction
- Scoring engine v0.1
- Map UI with country heat colors
- Alert thresholds

## Phase 2 (Calibration)
- Backtesting on historical conflicts
- Weight tuning and false-positive reduction
- Better confidence model

## Phase 3 (Expansion)
- More countries and subnational detail
- Scenario explorer (best/base/worst)
- Public API and reporting packs
